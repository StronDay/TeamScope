# Документация проекта

## Запущенные сервисы

| Сервис | Порт | Описание | URL (если есть) |
|--------|------|----------|------------------|
| Doxygen | `8000` | Документация кода | http://localhost:8000 |

2. Откройте браузер и перейдите по нужному URL
